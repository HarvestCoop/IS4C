Detailed Description

Handle standard manufacturer coupons

This module looks for UPCs with prefix

  • 005 UPC-12 coupon
  • 099 EAN-13 coupon

It extracts the manufacturer prefix from the UPC and validates that a matching item is in the transaction

It looks up the coupon code to calculate a discount value and adds the coupon to the transaction
