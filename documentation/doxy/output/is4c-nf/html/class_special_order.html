Detailed Description

WFC Electronic Special Orders

Special order upc format: prefix orderID transID 00454 xxxxxx xx

e.g., orderID #1, transID #1: 0045400000101

These IDs are used to locate the special order record in the PendingSpecialOrder table on the server database
